Smart Safety Detection System

This project focuses on detecting abnormal human motion using accelerometer data. The system analyzes magnitude values derived from x, y, and z coordinates and applies multiple conditions such as duration filtering, inactivity detection, and spike analysis to determine whether an emergency situation has occurred.

Problem Statement

Existing systems rely only on threshold-based detection which leads to false alarms. This system improves reliability by combining multiple logical conditions to identify real abnormal patterns instead of isolated spikes.

Methodology

The system processes accelerometer dataset values and computes magnitude. A sliding window approach is used to simulate real-time monitoring. Detection is based on duration filtering, inactivity detection, and pattern recognition.

Innovation

The system uses a hybrid logic combining multiple parameters rather than relying on a single threshold. This reduces false positives and improves detection accuracy.

Results

The system successfully differentiates between normal and abnormal activities. Controlled anomaly testing was used to validate detection due to limited availability of real-world emergency datasets.
