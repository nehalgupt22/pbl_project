Smart Safety Detection System

This project detects abnormal human motion using accelerometer data. Magnitude is calculated from x, y, z values and analyzed using duration filtering, inactivity detection, and spike patterns.

Problem Statement

Traditional systems rely only on threshold-based detection, which leads to false alarms. This project improves reliability by combining multiple logical conditions to detect real abnormal patterns.

Literature Review

Fall detection systems are widely used in healthcare and elderly monitoring. Most systems rely on accelerometer data and simple threshold methods, which can lead to inaccuracies. Our system improves this by incorporating duration and inactivity-based logic.

Innovation

This system uses hybrid detection combining multiple parameters rather than a single threshold, reducing false positives and improving detection accuracy.

Methodology

Dataset

Real accelerometer dataset from Kaggle is used for processing motion values.

Processing

Magnitude is calculated from x, y, z values and analyzed using sliding window logic.
