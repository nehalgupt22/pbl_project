Hybrid SOS Trigger Analysis System

Improving reliability of automatic SOS trigger mechanisms using temporal validation and multi-condition motion analysis.

Problem Statement

Wearable SOS systems rely on acceleration threshold-based triggering, which fails to distinguish between normal motion and emergency situations. Activities such as running or jumping generate short-duration spikes that exceed thresholds, causing false alerts. These repeated false triggers reduce reliability and may lead to alarm fatigue. Therefore, improving detection accuracy without increasing hardware complexity is necessary.

Literature Review

Mubashir et al. (2013) studied fall detection systems in healthcare environments such as hospitals and elderly care. They found that threshold-based systems are simple but produce high false positives during daily activities. Li et al. (2019) introduced temporal filtering, showing that sustained acceleration is more reliable than isolated spikes. Advanced machine learning-based systems improve accuracy but require high computation, making them unsuitable for embedded systems. This project adapts these concepts by introducing duration-based validation, inactivity detection, and repeated spike analysis while maintaining low computational cost.

Research Gap

Most embedded SOS systems still rely on simple threshold-based triggering and fail to validate motion over time. This project introduces a multi-layer detection mechanism to improve reliability.

System Methodology

Dataset

Real human activity dataset from Kaggle representing motion patterns.

Model

Threshold-based detection enhanced with duration filtering to detect sustained abnormal motion.

Advanced Logic

Includes inactivity detection after impact and repeated spike analysis to detect unstable conditions.

Results & Analysis

Threshold-only detection produced false alerts during short-duration spikes such as jumping. Duration-based filtering eliminated these false positives. Additional validation using inactivity and repeated spike patterns further improved detection accuracy and reliability.
