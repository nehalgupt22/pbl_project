Smart Safety Detection System

This project detects abnormal human motion using accelerometer data. Magnitude is calculated from x, y, z values and analyzed using duration filtering, inactivity detection, and spike patterns.

Problem Statement

Single-threshold systems generate false alarms. This project improves detection by combining multiple conditions to identify real abnormal patterns.

Literature Review

Existing fall detection systems use accelerometer and gyroscope data in healthcare applications such as elderly monitoring. Many rely only on threshold-based methods, which can lead to inaccurate detection. Our approach improves upon this by incorporating duration filtering and inactivity-based analysis.

Methodology

The system processes real dataset values and computes magnitude. A sliding window approach is used to simulate real-time monitoring. Detection is based on sustained spikes, post-event inactivity, and repeated abnormal patterns.

Innovation

The system uses hybrid detection combining multiple parameters rather than relying on a single threshold. This reduces false positives and improves reliability.

Results

The system differentiates between normal and abnormal motion. Due to limited availability of real emergency datasets, controlled anomaly testing was used to validate system behavior.
