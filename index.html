Smart Safety Detection System

This project detects abnormal human motion using accelerometer data. It combines magnitude analysis, duration filtering, inactivity detection, and spike patterns to identify emergency situations.

Problem Statement

Traditional systems rely only on threshold-based detection, leading to false alarms. This project improves reliability by combining multiple logical conditions to detect real abnormal patterns.

Literature Review

Fall detection systems are widely used in healthcare and elderly monitoring. Most rely on accelerometer data and simple threshold methods, which often lead to inaccurate detection. This system improves upon existing approaches by incorporating duration filtering and inactivity detection, making detection more reliable and robust.

Innovation

The system uses hybrid detection combining multiple parameters instead of a single threshold. This reduces false positives and improves detection accuracy significantly.

Methodology

Dataset

Real accelerometer dataset from Kaggle used for motion analysis.

Processing

Magnitude is computed from x, y, z values and analyzed using sliding window logic.
